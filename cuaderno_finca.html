Paso 1 — Crear proyecto en Google Cloud

Google Cloud es la plataforma de Google que permite usar sus APIs. Crear un proyecto es gratis y solo se hace una vez.

📋 Instrucciones (5 minutos)

  1. Abre console.cloud.google.com en una pestaña nueva
  2. Si es tu primera vez, acepta los términos de servicio
  3. Arriba a la izquierda verás un selector de proyectos — haz clic y luego en "Nuevo proyecto"
  4. Ponle un nombre como Vinedo-CampoCuaderno y pulsa Crear
  5. Espera unos segundos hasta que aparezca la notificación de "Proyecto creado"
  6. Asegúrate de tener el nuevo proyecto seleccionado en el selector de arriba
📍 Dónde estás: console.cloud.google.com · El selector de proyecto está arriba a la izquierda, junto al logo de Google Cloud.

🔌 Habilitar la Sheets API

  1. En el menú izquierdo, ve a APIs y servicios → Biblioteca
  2. Busca Google Sheets API
  3. Haz clic en el resultado y luego en el botón azul Habilitar
  4. Repite para Google Drive API (necesaria para crear la hoja)
Abrir Google Cloud Console →

Paso 2 — Crear credenciales OAuth

Las credenciales permiten que este HTML se identifique ante Google al conectarse.

🔑 Crear el Client ID

  1. En Google Cloud, ve a APIs y servicios → Credenciales
  2. Haz clic en + Crear credenciales → ID de cliente de OAuth 2.0
  3. Si te pide configurar la "Pantalla de consentimiento", haz clic ahí primero:
    • Tipo de usuario: Externo → Crear
    • Rellena solo el nombre de la app (p.ej. Cuaderno Viñedo) y tu email
    • Guarda y vuelve a Credenciales
  4. Tipo de aplicación: Aplicación web
  5. Nombre: Cuaderno Campo
  6. En "Orígenes autorizados de JavaScript" añade:
    null  (para archivos locales abiertos desde el disco)
  7. Haz clic en Crear
  8. Copia el Client ID (termina en .apps.googleusercontent.com)
⚠ Importante: El origen null es necesario porque abres el HTML como archivo local (file://). Sin esto, Google rechazará la conexión.
● Sin configurar
Abrir Credenciales →

Paso 3 — Conectar con tu cuenta Google

Pulsa el botón y se abrirá una ventana de Google para autorizarte. La primera vez puede pedir verificación porque la app está en modo desarrollo.

ℹ️ Si ves "Google no ha verificado esta app"

  1. Es normal — tu app está en modo de prueba
  2. Haz clic en "Opciones avanzadas" (o el enlace pequeño abajo)
  3. Luego en "Ir a [nombre de tu app] (no seguro)"
  4. Acepta los permisos solicitados

Paso 4 — Activar WhatsApp (CallMeBot)

CallMeBot es un servicio gratuito que envía mensajes a WhatsApp. Se activa en 2 minutos desde tu móvil.

📱 Activar desde el móvil (2 minutos)

  1. Abre WhatsApp en tu móvil
  2. Añade este número como contacto: +34 644 59 76 25
  3. Envíale exactamente este mensaje:
    I allow callmebot to send me messages
  4. En pocos segundos recibirás un mensaje con tu API Key (número de 6-7 cifras)
📍 Tip: Si no recibes la API Key en 1 minuto, intenta enviar el mensaje de nuevo. A veces tarda un poco la primera vez.
● Sin configurar
